Empowering Familial Genetic Research Through Data Visualizations

The Problem

Genetic Researchers want a better tool for efficiently analyzing familial data.
Other tools require a learning curve, are difficult to setup, and do not easily handle comparing genetic data between multiple individuals.

Spyro is here to help!

Try Spyro!

Spyro's Features

Multi-Family View

Researchers can easily pick the families from which they wish to view individual members.

Search by Index

Points of interest can be investigated by entering specific start and end coordinates.

BioGraph Visualization

Users can visualize and compare their data side-by-side by using Spiral Genetics’ uniquely powerful format for genomic data.

Save Session

Keeps track of which individuals the user is comparing to help them easily pick up where they left off.

Download

Downloads a comparison of the selected individuals’ BioGraph visualizations.
